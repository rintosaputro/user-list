# User List App

## How to run

- Clone this repository

```
git clone https://github.com/rintosaputro/user-list.git
```

- Go to folder repository

```
cd user-list
```

- Install modules

```
npm install
```

- Run app

```
npm run dev
```

- If the browser does not open automatically then open the browser and go to http://localhost:5173
- Done
